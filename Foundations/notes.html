Lists

Unordered lists

  • Created using ul
  • Within ul, each item is created using li
  • Each item in an unordered list beings with a bullet point

Ordered lists

  1. Created using the ol element
  2. Each individual item is also created with li
  3. Each list item in an ordered list begins with a number

Anchor elements

To create a link in HTML, wrap text/element which will be a link with <a>

  • Use an HTML attribute (href) to tell the anchor tag its destination

HTML Attributes

  • Gives additional information to an HTML elements

  • Without the target attribute, the link will open in the same tab as the webpage containing it
  • The target attribute specifies where the linked resource will be opened

  • Include the rel attribute for security purposes

rel

  • rel describes the relation between the current page and the linked document
  • noopener value ensures a link opened in a new tab/window cannot interact with the original page
  • noreferrer provides both privary and security by hiding the referrer from the new page

Absolute and relative links

Absolute Links: links to pages on other websites on the internet

  • scheme://domain/path
  • scheme: http
  • domain:www.theodinproject.com
  • path:about

Relative Links: links to other pages within our own website

  • does not include the domain name
  • only includes the file path to the other page relative to the page the link is being created on
  • add ./ to specify that your code should start looking for the file/directory relative to the current directory

Images

  • The <img> is a void element
  • Instead of wrapping content, it embeds an image into the page using a src attribute
  • src attribute tells the browser where the image file is located
  • src attribute can use both absolute and relative paths as well

Alt attribute

  • Every image element should also have this attribute
  • Used to describe an image
  • Used in place of the image if it cannot be loaded
